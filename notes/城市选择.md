# 城市选择

## 路由的使用
```text
- 修改src/router/index.js文件
routes: [
  {
    path: '/',
    name: 'Home',
    component: Home
  },
  {
    path: '/city',
    name: 'City',
    component: City
  }
]

- 使用router-link标签
<router-link to="/city">
但是要注意使用这个标签相当于添加了一个a标签，这样会导致字体变成其他颜色，需要自己添加color:#fff样式
```
